- Android SDK (Android Studio recommended)

### **Build Steps:**
1. **Clone repository:**
   ```bash
   git clone https://github.com/ether4o4/4GAuteauOS.git
   cd 4GAuteauOS
   ```

2. **Run build script:**
   ```bash
   chmod +x build_local.sh
   ./build_local.sh
   ```

3. **Find APK:**
   - Location: `app/build/outputs/apk/debug/app-debug.apk`

## **🔧 Troubleshooting:**

### **If APK won't install:**
1. Check Android version (requires 8.0+)
2. Enable "Install from unknown sources"
3. Try downloading again (corrupted download)

### **If launcher doesn't appear:**
1. Go to Android Settings → Apps
2. Find "4GAuteauOS"
3. Set as default home app

### **If build fails:**
1. Ensure Java JDK 17+ is installed
2. Install Android Studio for SDK
3. Check error messages in terminal
